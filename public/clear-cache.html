🧹 Clear All Caches

This page will help you clear all caches, including stubborn 401 responses.

What this does:

  • Clears all Cache Storage
  • Clears IndexedDB
  • Clears Local Storage
  • Clears Session Storage
  • Unregisters all Service Workers
  • Clears Cookies (same-origin)

Manual alternative:

  1. Open DevTools (F12)
  2. Go to Application tab
  3. Click "Clear site data"
  4. Or use Incognito mode
