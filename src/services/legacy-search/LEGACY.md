# Legacy Search (deprecated)

This is the original Search service, kept temporarily for the `/search`,
`/avatar-search`, and Pro/ProTest routes. It has been **superseded by the new
Search** (formerly "Fast Search") under [`src/services/search/`](../search/),
which is now the default mode in the search box.

**Status:** slated for removal. Do not build new features on top of this.
Once the routes/links that still point here are migrated or retired, this whole
directory can be deleted.
